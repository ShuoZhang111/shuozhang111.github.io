ERROR 1698 (28000) Access denied for user 'root'@'localhost'

更新于 2018-10-13

最近在电脑上安装新Mysql是遇到了一个ERROR 1698 (28000): Access denied for user ‘root’@’localhost’问题
因为我使用的是Ubuntu，mysql默认使用的是UNIX auth_socket插件
意味着：db_users使用系统用户凭证”auth”我们可以通过下面的方式判断我们是不是真的这样

如果查询中看到root用户正在使用auth_socket插件
我们有两种办法解决它：

  • （一）设置root用户使用mysql_native_password插件
  • （二）可以创建一个新的system_user
解决（一）

在新的mysql中，如果密码在安装时保留为空，则它基于auth_socket插件
正确的方法是使用sudo权限登陆Mysql
sudo mysql -u root -p

然后使用下面命令更新密码
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'new-password';

完成后重启mysql
sudo service mysql stop
sudo servide mysql start

解决（二）

完成后重启mysql
sudo service mysql stop
sudo servide mysql start
