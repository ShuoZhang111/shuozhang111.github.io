关于二进制的一些认识

更新于 2018-11-12

二进制是一种逢二进一的计数规则

补码

补码是一种计算机中用于处理有符号数的一种编码规则，其思想是将有限位数的二进制数一分为二，以最高位作为标志位，对于Java语言来说最高位为1则表示的数位负数。

四位补码

  • 互补对称性
    • (-n) = (~n + 1) （最小值除外）负n = 取反 n + 1 （最小值除外）

二进制的计算

与运算 &

  • 按位对齐，全1得1，有0得0

    • 例如： 10001111

  • 运用1：掩码Mask计算,截取数据

    • 例如：对10001111 10001111 进行8位掩码计算

    这样我们就截取到了10001111 10001111这组数据的后八位数据，这在网络数据传输中应用广泛

  • 运用2：优化对2的整数次幂取余的优化

    • 例如： 123224 & 8; 我们可以优化为 123224 & 7；
    • 123224 & 16；我们可以优化为 123224 & 0xf;
右移位计算 >>>
  • 将二进制数据的整体向右移动，低位自动溢出舍弃，高位补0；
    • 例如：01101001 11110101 01011111 00101001 >>> 8 = 00000000 01101001 11110101 01011111
  • 运用1 配合 掩码Mask运算 假设我们在数据传输中的大小无法适用于我们原始数据的大小，我们使用以为和掩码Mask运算对原始数据的二进制数进行截取，分开传输。
左移位 <<
  • 将二进制数字整体向左移动，高位溢出舍弃，低位补0
    • 例如:01101001 11110101 01011111 00101001 << 8 = 11110101 01011111 00101001 00000000
或运算 |
  • 将二进制数按位上下对齐；遇到1就是1；都是0就是0
    • 例如: 00000000 00000000 10011010 00000000 | 00000000 00000000 00000000 11111111 = 00000000 00000000 10011010 11111111
  • 运用1:左移位可以将我们方便数据传输是所截取的二进制数字进行合并
    • 例如: 00000000 00000000 00000000 10011010 << 8 = 00000000 00000000 10011010 00000000

移位计算数学意义

  • 我们在10进制中向左移动以为是扩大10倍,当然我们在2进制中同样适用
<<
  • 数字向左移动，数字每移动一次扩大2倍。
    • 例如: n = 00000000 00000000 00000000 00110010 50
    • m = n << 1 0000000 00000000 00000000 001100100 100
    • k = m << 1 000000 00000000 00000000 0011001000 200
    • g = k << 1 00000 00000000 00000000 00110010000 400
数学右移位 >>
  • 每移动一次数字其结果相当于数字除以2，向小方向取整
    • n = 00000000 00000000 00000000 00110010 50
    • m=n>>1 000000000 00000000 00000000 0011001 25
    • k=n>>2 0000000000 00000000 00000000 001100 12
右移位 >> 和 >>> 的区别
  • 正数时候的运算结果一样
  • 负数的时候
    • >> 高位补1,保持数学除法运算结果
    • >>> 高位补0,就是逻辑上数字向右移动,不是数学中的除法

面试题目

  • System.out.println(~-9);
    • 正确答案是会 ~-9 = 8,(补码的对称性: -n = ~n + 1)
  • 一个正数溢出以后是负数?
    • 正确答案是:不一定,溢出有可能是负数,有可能是正数,还有可能会是它本身
    • 例如: m = 10 + Integer.MAX_VALUE + 1 + Integer.MAX_VALUE + 1;结果m = 10;溢出会他本身

Eclipse Console不显示

更新于 2018-08-16

我们在使用Eclipse边运行项目边写代码的时候难免会受到Console弹窗的烦恼
在Preferences->Run/Debug->Console里面,有两个是关于显示的：

  • 一个是Show when program writes to standard out(当Console中有值弹出)
  • 一个是Show when program writes standard error(当Console中有错误是弹出)。

我们把这两个取消勾选就可以了。

贝济埃曲线

贝济埃曲线（EN:Bezier Curve 即贝塞尔曲线）是一些曲线几何的总称。 在本文中主要像大家展示三种Bezier Curve分别是Linear Bazier curves Quadratic Bezier curves Cubic Bezier curves 这三种都是经常使用的三种曲线，分别是一次曲线、二次曲线和三次曲线，当然还有一些更高维度的曲线就省略了，最后我会放几个图片大家自行理解一下。

Linear Bazier curves

一次Bazier Curve 很容易理解，它实际上指的就是一条直线，没有任何弯度，所以使用一次Bazier Curves 进行插值的技术，有被称为Linear Interpolating。线性插值技术使用还是很广泛的，除去动画系统中，在3D图形的一些滤镜处理阶段也有使用，像光栅化阶段也用来对顶点数和纹理数据进行插值计算。
Linear BazierB(t) = p0 + (p1 - p0) * t = (1 - t)p0 + t * p1,t in [0,1]
给定点p0 p1,而B(t)为此时曲线的状态。此时是直线。
