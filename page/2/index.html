Flask 学习笔记（一）

发表于 2017-09-21 | 分类于 Python

其实本文与其说是Flask 学习笔记不如说是Flask Web开发基于Python的Web应用开发实战的读书笔记

Flask简介

在大多数标准中Flask都算是一个小型框架，但是小并不意味者它比其他框架功能少。

Flask有两个主要依赖 Werkzeug和jinja2这两个依赖都是由Flask核心开发者开发而成：

  • 路由，调试，和Web服务器网络接口(Web Server Gateway Interface,WSGI)子系统由Werkzeug提供。

  • 模块系统由Jinja2提供。

Flask并不原生支持数据库访问，Web表单验证和用户认证等高级功能。这些功能以及其他大多Web程序中需要的核心功能都是以拓展形式实现。

Flask安装

既然Flask这么自由可定制那么我们是如何使用它的呢?

  • virtualenv 虚拟环境是非常有用的，可以在系统的Python解释器避免包的混乱和版本冲突。为每个程序创建单独虚拟环境。
    • 安装
      • Linux(Ubutu) sudo apt-get install python-virtualenv
      • Windowns easy-install virtualenv
    • 使用
      • cd 到对应文件夹 virtualenv --no-site-packages {name}即可创建没有任何拓展包的虚拟环境
      • Linux运行 source {name}/bin/activate
      • Windows {name}/bin/activate
  • 使用pip安装Python包
    • (env) $ pip install flask
      • (env) $ python
      • >>> import flask
      • >>>
    • 没有错误提示即说明安装成功。

      Flask使用

  • 所有Flask程序都必须构建一个程序实例。Web程序使用一种Web服务器网关接口（WSGI）的协议，把接收自服务器的所有请求都交给这个对象处理。

    from flask import Flask
app = Flask(__name__) 
@app.route('/')
def index():
    return '<h1>hello world!</h1>'

  • (env) $ python name.py

  • 打开提示网络地址和端口默认的是http://127.0.0.1:5000

比你努力的人比你更优秀，比你优秀的人比你更努力

发表于 2017-09-11 | 分类于 其他

聪明干不过勤奋，勤奋干不过基因。

如果用数学方式将它排一下的话……

聪明<勤奋<基因。

既然基因这事儿是我们无法决定的，那么唯一的决定因素便是勤奋。

说起这事儿，我容易想到的一个例子便是我的一个亲戚。他跟我年龄相差不大，所以对于我而言，我更容易了解他。他的学习生活完全可以用一句话概括：英语从小学差到高中，不出意料的话，大学也是如此。但最近，他倒是有发展英语的趋势，可惜只不过是“三天打鱼两天晒网”。

我们班上成绩好的一个女生，戴着800度的眼睛，离了眼镜就什么也看不见了。可是不能否认她的成绩好，她是通过题海战术才提起来成绩的。老师说：“有量才有质。”恕我不能认同。她才十几岁，就用一副眼镜葬送了她的前半生与后半生。是的，她比我要优秀，她比我要努力。但是在此我只想说努力是要偷偷的，不为人所知。你要是知道了，被人了解到你比他要更努力，那岂不是老底都被摸清了吗？

什么是优秀？超出别人所期料即优秀。

有人认为优秀就是你做到最好，这只是一方面，最重要的是别人对你是什么标准，而你超过了这个标准。

这像是跳高，每次跳都是在跳过你的极限。拿学习来说，超前学习就是优秀；工作，认真且仔细地完成。哦不，我忘了工作学习高效才是真牛。

我也不能否认除了勤奋更有其他的决定因素在其中，身份背景都会对此产生影响。可是毕竟勤奋能为你得到更多的资源，就凭这点，在这个只有一块蛋糕的世界才能争取到更多。

虽然我认为我说这么多的全是些废话：

但是，你要永远记住这句话：比你努力的人比你更优秀，比你优秀的人比你更努力。

这无疑是一句很好的自勉的座右铭。
